# 👋 你好! 这里是山西农业大学嵌入式实验室

🧑 一个新成立的实验室，隶属于山西农业大学软件学院

🌱 保持学习，希望在有限的时间内，学到无限的可能

🐛 实验室注重人才的培养，具有自己的培养体系

💡 欢迎每一个志同道合的人加入，需要具备自学能力和一定的编程能力


### 实验室福利

- **独立的场地** 实验室给每个成员配备自己的工位，能够让你长时间投入的学习。

- **督促学习** 实验室具有自己的培养方案，让你的大学学习不会迷茫，争取成为追逐时代的计算机人才。

- **竞赛福利** 实验室引导项目制作，独立完成每一个项目，期间经费有报销余额。

### 实验室技术栈

- C++
